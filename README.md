# nestjs-ts-postgresql-swagger-interceptor

Built with [Nestjs](https://github.com/nestjs/nest), an Express.js framework, in combination with [PostgreSQL](https://www.postgresql.org/).

## Install

```bash
pnpm install
```

## Run

```bash
# development
pnpm run start

# watch mode
pnpm run start:dev

# production mode
pnpm run start:prod
```

## Test

```bash
# unit tests
pnpm run test

# e2e tests
pnpm run test:e2e

# test coverage
pnpm run test:cov
```
